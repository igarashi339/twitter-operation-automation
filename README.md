## 開発環境立ち上げ手順
1. コンテナを起動する
    ```
    make up
    ```
1. コンテナのシェルにはいる
    ```
    make shell
    ```
1. claspログインする
    ```
    clasp login --no-localhost
    ```
1. ターミナルに表示されたURLに行く
1. 認証を進めると認証コードが表示されるので、コードをコピーしてターミナルにペーストする
1. おわり

## プッシュ手順
1. ビルドする
    ```
    npm run build
    ```
1. プッシュする
    ```
    clasp push
    ```
